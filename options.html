📑 Bookmark Categories

🎯 How Organization Works

This extension uses AI to intelligently sort your bookmarks into folders. You have two approaches:

  • Create Folders: AI creates new folders based on your predefined categories below. This ensures consistent folder structure and prevents the AI from inventing new folders each time, which would result in chaos.
  • Use Existing Folders: AI organizes bookmarks into your existing folder structure. Bookmarks that don't fit well will be left in their current location rather than forced into an inappropriate folder.

💡 Tip: Pre-defined categories maintain organizational consistency across reorganizations, preventing endless folder proliferation and restructuring.

One category per line. These categories will be used to organize your bookmarks. Add, remove, or rename categories to match your preferences.

The extension remembers which bookmarks it has organized to avoid moving them again unnecessarily. Use these settings and tools to control that behavior.

⏱️ Performance Settings

seconds
Maximum time to wait for AI API responses. Increase if you have a slow connection. (30-600 seconds)
bookmarks
Number of bookmarks to process in each batch. Smaller batches are faster but may create inconsistent categories. (10-100 bookmarks)
tokens
Maximum tokens for AI response. Higher values allow processing larger batches but cost more. Rate limiting is handled automatically with smart retry logic. (1024-8192 tokens)
